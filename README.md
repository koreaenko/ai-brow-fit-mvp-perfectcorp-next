# AI Brow Fit MVP

모바일 우선 눈썹 가상 상담 웹앱입니다. 사진을 올리면 브라우저에서 MediaPipe Face Landmarker로 얼굴 특징을 감지하고, 눈 위치와 얼굴 기울기를 기준으로 SVG/path 스타일의 Canvas 눈썹 템플릿을 자동 배치합니다.

## 기능

- 사진 업로드 및 모바일 카메라 입력
- 브라우저 기반 얼굴 랜드마크 감지
- 눈 안쪽/바깥쪽, 눈 사이 거리, 얼굴 기울기 기반 자동 눈썹 배치
- 6개 눈썹 스타일
- 아치, 두께, 길이, 높이, 간격, 진하기, 선명도 조정
- 좌우 대칭 보정
- 자동 맞춤 다시 실행
- 원본/적용 전후 비교
- PNG 결과 저장

눈썹 색상 선택은 MVP 범위에서 제외되어 있으며, 자연스러운 다크브라운/차콜 색상으로 고정됩니다.

## 실행

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:3000`을 열면 됩니다.

## 배포

Vercel에서 일반 Next.js 프로젝트로 배포할 수 있습니다. 별도 API Key는 필요하지 않습니다.

## 참고

MediaPipe 모델과 WASM 파일은 브라우저에서 공개 CDN으로 로드됩니다. 서버로 얼굴 사진을 업로드하지 않고, 편집과 PNG 저장은 클라이언트 Canvas에서 처리합니다.
